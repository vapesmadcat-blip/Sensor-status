## 🔧 Instalação

### Requisitos
- **Android 7.0 ou superior**
- **Espaço disponível**: Mínimo 50MB
- **Permissões necessárias**: Acesso a localização e armazenamento

### Passo a Passo

1. **Baixe o arquivo APK** do Location Cleaner
2. **Abra o arquivo** no seu celular (geralmente em Downloads)
3. **Clique em "Instalar"** quando solicitado
4. **Aguarde a instalação** completar
5. **Clique em "Abrir"** para iniciar o app

> **Nota**: Se receber um aviso de segurança, é normal. O app é seguro e não coleta dados pessoais.

## 🏠 Tela Principal

Quando você abre o Location Cleaner, você vê a tela principal com os seguintes elementos:

### Cabeçalho
- **Título**: "Location Cleaner"
- **Descrição**: "Gerencie seu histórico de localização e sensor de umidade"

### Dois Cartões Principais

#### 🗑️ Cartão "Limpar Cache USB" (Azul)
- **Ícone**: Lixeira com linhas
- **Função**: Remove todo o histórico de localização armazenado

- **Exemplo**: "5 registros armazenados"

#### 💧 Cartão "Resetar Umidade" (Verde)
- **Ícone**: Gota de água
- **Função**: Reinicia o sistema de detecção de umidade
- **Status**: Mostra o estado atual do sensor
- **Exemplo**: "Status: Não calibrado" ou "Status: Ativo ✓"

### Seção de Histórico de Localização
- **Título**: "Histórico de Localização"
- **Botão "Ver Tudo"**: Expande para visualizar todos os registros
- **Filtros**: Permite filtrar por tipo de localização
- **Preview**: Mostra os 3 primeiros registros

## ⚙️ Funcionalidades Principais

### 1️⃣ Limpar Cache USB (Apagar Tudo)

**O que faz**: Remove completamente todo o histórico de localização do seu celular.

**Como usar**:
1. Clique no cartão azul **"Limpar Cache USB"**
2. Uma janela de confirmação vai aparecer
3. Leia o aviso: "Você tem certeza que deseja apagar X registros de localização?"
4. Clique em **"Apagar Tudo"** para confirmar (botão vermelho)
5. Ou clique em **"Cancelar"** para desistir
6. Se confirmado, você verá a mensagem: "Todos os registros foram apagados!"

**⚠️ Atenção**: Esta ação é **irreversível**. Todos os dados serão deletados permanentemente.

### 2️⃣ Resetar Sensor de Umidade

**O que faz**: Reinicia o sistema de detecção de umidade do seu dispositivo.

**Como usar**:
1. Clique no cartão verde **"Resetar Umidade"**
2. Uma janela de confirmação vai aparecer
3. Leia o aviso: "Você tem certeza que deseja resetar o sensor de umidade?"
4. Clique em **"Resetar"** para confirmar (botão vermelho)
5. Ou clique em **"Cancelar"** para desistir
6. Se confirmado, você verá: "Sensor resetado! Último reset: Agora"

**💡 Dica**: Use esta função se o sensor estiver apresentando leituras incorretas.

### 3️⃣ Filtrar Histórico de Localização

**O que faz**: Permite visualizar apenas registros de um tipo específico de localização.

**Tipos de Localização**:
- **GPS**: Localização precisa via satélite
- **WiFi**: Localização aproximada via rede WiFi
- **Célula**: Localização via torre de celular
- **Todos**: Mostra todos os registros

**Como usar**:
1. Na seção "Histórico de Localização", você verá 4 botões de filtro
2. Clique no botão do tipo que deseja filtrar
3. O botão selecionado fica **azul**
4. A lista de registros atualiza automaticamente
5. Clique novamente para desselecionar

**Exemplo**: Se clicar em "GPS", verá apenas registros de localização por GPS.

## 📍 Gerenciamento de Histórico

### Visualizar Histórico Completo

**Como acessar**:
1. Clique no botão **"Ver Tudo"** na seção "Histórico de Localização"
2. A tela vai expandir mostrando o histórico completo
3. Você verá uma lista com todos os registros

### Estrutura de Cada Registro

Cada registro de localização mostra:
- **Checkbox**: Para selecionar o registro
- **Ícone**: Indica o tipo (GPS 📍, WiFi 📶, Célula 📡)
- **Endereço**: Local da localização registrada
- **Tipo e Tempo**: "GPS • 1h atrás" ou "WiFi • 2d atrás"

### Selecionar Registros Individuais

**Como fazer**:
1. Clique no **checkbox** (quadrado) ao lado do registro
2. O checkbox fica **azul** quando selecionado
3. Um **checkmark** (✓) aparece dentro
4. Você pode selecionar múltiplos registros

**Selecionar Todos**:
1. Na tela de histórico completo, clique em **"Selecionar Todos"**
2. Todos os registros ficarão marcados
3. Clique novamente para desmarcar todos

### Deletar Registros Selecionados

**Como fazer**:
1. Selecione um ou mais registros (veja acima)
2. Um botão **"Deletar X Selecionado(s)"** aparecerá em vermelho
3. Clique no botão vermelho
4. Uma confirmação aparecerá: "X registros deletados!"
5. Os registros serão removidos da lista

**⚠️ Atenção**: Esta ação também é **irreversível**.

### Voltar para Tela Principal

**Como fazer**:
1. Clique no **ícone X** (fechar) no canto superior direito
2. Ou use o botão voltar do seu celular
3. Você retornará à tela principal

## 💡 Dicas e Truques

### Dica 1: Limpeza Periódica
Recomendamos limpar o histórico de localização **a cada 2-4 semanas** para manter o dispositivo otimizado.

### Dica 2: Filtrar Antes de Deletar
Se você quer deletar apenas registros de um tipo específico (ex: apenas WiFi), use o filtro primeiro. Assim você vê exatamente o que vai deletar.

### Dica 3: Verificar Antes de Apagar Tudo
Sempre clique em "Ver Tudo" antes de usar "Limpar Cache USB" para ter certeza de quantos registros serão deletados.

### Dica 4: Sensor de Umidade
Se seu celular mostrar leituras estranhas de umidade, tente resetar o sensor. Isso geralmente resolve problemas de calibração.

### Dica 5: Feedback Tátil
O app vibra quando você toca nos botões. Se não sentir vibração, verifique se o vibrador está ativado nas configurações do seu celular.

## ❓ Perguntas Frequentes

### P: O app coleta meus dados de localização?
**R**: Não. O Location Cleaner **não coleta, não armazena e não envia** seus dados para ninguém. Ele apenas gerencia os dados que já estão no seu celular.

### P: Posso recuperar dados deletados?
**R**: Infelizmente, não. Uma vez deletado, o histórico é removido permanentemente. Por isso, sempre verifique antes de clicar em "Apagar Tudo".

### P: O app funciona sem internet?
**R**: Sim! O Location Cleaner funciona **completamente offline**. Você não precisa de conexão com internet para usar.

### P: Quanto espaço o app usa?
**R**: O app usa menos de 50MB de espaço. O histórico de localização é armazenado no próprio dispositivo.

### P: O que significa "Não calibrado" no sensor de umidade?
**R**: Significa que o sensor ainda não foi calibrado. Clique em "Resetar Umidade" para calibrar.

### P: Posso deletar apenas alguns registros?
**R**: Sim! Use os checkboxes para selecionar registros específicos e clique em "Deletar Selecionados".

### P: O app funciona em tablets?
**R**: Sim, o app funciona em qualquer dispositivo Android 7.0 ou superior, incluindo tablets.

### P: Como faço backup do histórico?
**R**: Atualmente, o app não possui função de backup. Os dados são salvos apenas no dispositivo.

### P: Há limite de registros que posso armazenar?
**R**: O app pode armazenar até 1.000 registros de localização. Depois disso, os mais antigos são automaticamente removidos.

## 🆘 Solução de Problemas

### Problema: O app não abre
**Solução**:
1. Reinicie seu celular
2. Desinstale e reinstale o app
3. Verifique se tem espaço disponível no dispositivo

### Problema: Os registros não aparecem
**Solução**:
1. Verifique se o filtro está em "Todos"
2. Tente fechar e abrir o app novamente
3. Verifique se você tem permissão de acesso à localização

### Problema: O botão "Deletar" não funciona
**Solução**:
1. Certifique-se de ter selecionado pelo menos um registro
2. Tente selecionar novamente
3. Reinicie o app

### Problema: A vibração não funciona
**Solução**:
1. Verifique se o vibrador está ativado nas configurações do celular
2. Aumente o volume do dispositivo
3. Tente em outro app para confirmar se o vibrador funciona
