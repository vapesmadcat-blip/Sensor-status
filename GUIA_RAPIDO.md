# ⚡ Guia Rápido - Location Cleaner

## 🚀 Começar em 30 Segundos

### Instalação
1. Baixe o APK
2. Abra o arquivo
3. Clique em "Instalar"
4. Pronto! ✅

---

## 🎯 3 Coisas Que Você Pode Fazer

### 1. Apagar Todo o Histórico
```
Clique no cartão AZUL "Limpar Cache USB"
→ Confirme
→ Pronto! Todos os registros foram deletados
```

### 2. Resetar Sensor de Umidade
```
Clique no cartão VERDE "Resetar Umidade"
→ Confirme
→ Pronto! Sensor foi resetado
```

### 3. Deletar Registros Específicos
```
Clique em "Ver Tudo"
→ Marque os registros com checkbox
→ Clique em "Deletar Selecionados"
→ Pronto! Registros foram removidos
```

---

## 🔍 Filtrar por Tipo

Clique em um dos botões:
- **Todos** - Mostra tudo
- **GPS** - Apenas GPS
- **WiFi** - Apenas WiFi
- **Célula** - Apenas Célula

---

## ⚠️ Lembre-se

- ✅ Funciona **sem internet**
- ✅ **Não coleta dados**
- ❌ **Deletar é irreversível**
- ❌ **Sem backup automático**
